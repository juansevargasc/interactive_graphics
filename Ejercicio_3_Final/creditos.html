Descripción del Proyecto

Este proyecto consiste en la creación de un blog con un mapa interactivo que muestra los principales concesionarios de autos en Colombia, infromación sobre las marcas principales para un segmento de automoviles muy importante como los 'Hot Hatch'.

Los elementos que componen el proyecto incluyen:

  • HTML: Estructura del contenido y la navegación del sitio web.
  • CSS: Estilos, diseño, wntro otros.
  • JavaScript: Utilizado para implementar la lógica de la interacción incluyendo la inicialización del mapa y los marcadores dentro del mismo.
  • Leaflet.js: Una biblioteca de JavaScript de código abierto utilizada para crear mapas. Leaflet permite agregar marcadores y popups, para porveer la información en este caso de concesionarios.

Mapa Interactivo

Cada marcador en el mapa proporciona información relevante sobre los concesionarios, incluyendo la marca, ubicación y un enlace al sitio web oficial del concesionario 🚙
